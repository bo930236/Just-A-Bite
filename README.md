Just-A-Bite 餐廳網站
============
[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://just-a-bite.vercel.app/ "前往 DEMO")

## Front Page
- 使用 CSS + RWD 手刻餐廳網站
- 介紹餐廳、特色菜單、評論、地圖顯示位置
- 點擊 navbar 可以前往其他頁，點擊 Just-A-Bite 回到首頁

![Just-A-Bite 餐廳網站 Preview2](https://img.onl/9xmU7)

## Lottery
- 使用 Javascript 完成抽獎功能
- 使用 API 串接獎項
- 點擊 "我要抽獎" 會顯示抽獎結果及圖片
